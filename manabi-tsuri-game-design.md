# まなびじまのつりぼうけん きほんせっけい

## もくてき

しょうがく1ねんせいのこどもが、さんすう・ローマじ・いきものにたのしくふれながら、できることがふえていくじっかんをもてるPCようがくしゅうゲームをつくる。

ゲームはこじんりようをまえ提にし、むりょうでかいはつ・うんようする。

## こうかい・うんよう

- GitHub Pagesでこうかいする。
- サーバーしょりはつかわない。
- HTML、CSS、JavaScriptだけでうごかす。
- しんちょくはブラウザのlocalStorageにほぞんする。
- ログイン、データベース、ランキングはさいしょはつくらない。

## せかいかん

タイトル案:

```text
まなびじまのつりぼうけん
```

主人公は、ちいさなみならいつりびと。  
さんすう・ローマじ・いきものをがんばると、主人公やアイテム、あいぼうがすこしずつそだつ。

つりはメインのべんきょうからはなした、ごほうび・コレクションようそにする。

## がくしゅうこうもく

```text
1. さんすう
2. ローマじ
3. いきもの
```

## せいちょうルール

```text
さんすう Lv1-10   -> 主人公のからだ・かみがた
ローマじ Lv1-10   -> 主人公のふく・アイテム
いきもの Lv1-10   -> あいぼう
つり              -> さかなずかん・コレクション
```

「ぶき」ではなく「アイテム」としてあつかう。  
こどもむけなので、あぶない印象のものはさける。

## クリアルール

- 1レベルは5もん。
- 5もんぜんぶせいかいでクリア。
- 1もんでもまちがえると、そのレベルはクリアにならない。
- なんどでもチャレンジできる。
- しっぱいしても、しからない。
- しっぱいしても「えさのかけら」を1こもらえる。
- えさのかけら3こで「つりけん」1まいにできる。
- クリアすると「つりけん」1まいをもらえる。

## ゲーム内テキストルール

ゲーム内に表示する文字は、こどもが自力で読めることを優先する。

```text
OK:
はじめる
もういちど
つりにいく
ずかん
できた！
おしい！
よくできたね！
せいかい！
ちがうよ

NG:
開始
挑戦
図鑑
成長
正解
不正解
装備
獲得
```

ルール:

- かんじはつかわない。
- ひらがな・カタカナだけにする。
- むずかしいことばはつかわない。
- ぶんはみじかくする。
- ボタンはひらがな中心にする。

## つりのしくみ

```text
がくしゅうをクリア
  -> つりけんをもらう
  -> つりばでつる
  -> さかなをゲット
  -> ずかんにのる
```

しっぱいしたとき:

```text
がくしゅうにチャレンジ
  -> クリアできない
  -> えさのかけらを1こもらう
  -> 3こでつりけん1まい
```

## さかな

さかなは、ずかんをうめるコレクションようそにする。

種類:

- よくでるさかな
- たまにでるさかな
- レアなさかな
- でんせつのさかな
- はずれ: ながぐつ、あきかんなど

## がぞうそざい方針

画像は生成をメインにする。

理由:

- えがらをそろえやすい。
- ライセンスかくにんの手間がすくない。
- Lv1-10のせいちょうをまとめて設計しやすい。
- あとから追加しやすい。

方向性:

- 2Dイラスト
- あかるい
- かわいい
- しょうがく1ねんせいむけ
- すこしゲームふう
- こわくない
- 背景はえほんふう
- キャラはパーツ合成しやすい正面立ち絵
- さかなは横向き
